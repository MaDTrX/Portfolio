NCAA SCHEDULES

This app was created with the goal of hosting all college sports schedules. To accomplish this, I created ncaaSchedules API. The REST API gets data from SportsDataIO API and by scraping the web. It then aggregates, sorts the gathered information, and stores them into a MongoDB updating the data at set intervals.

SAFIRI

A traveling app that allows users to plan their trip with ease. Keep your adventures organized by logging checklists, flight itenaries, hotel bookings, and budget planning. Powered by Amadeus API & Google Places API.

CHECKERS

An online version of the classic game of Checkers built using SocketIO and JQuery. The game implements a chat functionality for users who play against each other. The website features local and online player vs. player.

UP4GRUBZ

This full CRUD app was created with the mission to eliminate food waste. Users can login to post their extra and/or about-to-expire food so that other users can claim them. Everyone is welcome to browse a list of the posted grubs but they must be logged-in to post or claim grubs. The app utilizes the google places markers API to display a map with the location of posted grubs. Users can add and delete multiple photos of posted grubs. Photos are stored and deleted in S3 buckets based on user activity.
